# Aetheria AI - Modern Startup Landing Page

A premium, highly interactive, and fully responsive dark-themed landing page template designed for modern AI startups. Crafted utilizing semantic **HTML5**, high-fidelity **Vanilla CSS3**, and fluid **Vanilla JavaScript**.

---

## 🌟 Visual & Interactive Features

*   **Dark Navy & Neon Theme**: Elegant deep navy background accented with vibrant HSL-derived purple and cyan gradient combinations.
*   **Atmospheric Backdrop Glows**: Drift-animated glowing blur spheres pulsing gently behind elements to create rich, multi-layered visual depth.
*   **Vercel-Style Mouse Glow**: Grid feature cards that calculate and reflect the user's relative mouse coordinates in real-time, casting a soft radial highlight that tracks your cursor.
*   **3D Perspective Browser Frame**: A mock desktop browser containing the primary AI dashboard illustration. Hovering over it transitions the container smoothly from a rotated 3D angle into a head-on focus.
*   **Scroll-Spy Navigation Bar**: An automated navigation header that darkens, shrinks, and increases its backdrop blur as the viewport shifts, updating the active navigation item dynamically based on scroll location.
*   **Statistical Count-Up counters**: Automatically animates numerical metrics from `0` to their final values using a custom quadratic-out easing curve once the metrics section is scrolled into view.
*   **Section-Reveal Triggers**: Smooth scroll-triggered entrance animations driven by `IntersectionObserver` to reveal text structures and cards progressively.
